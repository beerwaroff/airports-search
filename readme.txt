Как работает программа:
1. Осуществляется сортировка выбранной колонки в целях бинарного поиска.
2. Вводится текст и запускается секундомер.
3. Происходит поиск нужных индексов методом бинарного поиска по данным, представляющим отсортированную колонку
(индексы составляют множество).
4. Файл airports.csv читается потоком без хранения информации в структуре данных, пока множество с индексами не пустое
(при нахождении индекс удаляется из множества).
5. Останавливается секундомер.

Файл читается до тех пор, пока не найдены все строки с нужными индексами.
Как только нужные строки найдены, программа завершает выполнение.

Таким образом, перечитываются не все строки файла.
